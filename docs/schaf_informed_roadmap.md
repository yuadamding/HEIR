# SCHAF-informed HEIR roadmap

## Decision principle

The transferable lesson is to qualify a stronger image-to-residual translator before increasing
reference-bank complexity. HEIR retains its donor-safe residual target and falsification controls;
it does not adopt a large end-to-end generator, adversarial alignment, or optimal transport merely
because those components exist in another architecture.

```text
Preserve frozen evidence
        |
        +-- Exposed HEST nonlinear qualification
        |       |
        |       +-- fail: stop neural expansion; improve measurement/representation
        |       |
        |       +-- pass: preregister prospective_nonlinear_v1
        |
        +-- Annotation provenance -> H-MEAS -> sequential calibration
                                        |
                                        +-- pristine H-CELL
                                                |
                                                +-- H-REF v3
                                                        |
                                                        +-- H-END2END / H-COMP
```

## Track A — estimator qualification on exposed data

1. **WP-1: frozen nonlinear protocol — complete.** Register B0–B3 and N0–N7, the complete support rule,
   architecture/training settings, permanent non-authorization, and the frozen predecessor hashes.
   Execution remains blocked until a receipt-bound blank-patch supplement and an evaluable
   preregistered registration sensitivity exist.
2. **WP-2: deterministic nonlinear residual probe — core complete.** Add tiny/small MLPs, a fine-type FiLM
   adapter, late fusion, nested donor selection, deterministic checkpoint receipts, and seed
   stability tests.
3. **WP-3: architecture-matched controls and refitted nulls — core complete.** Compare nonlinear image models with
   nonlinear nuisance, blank-patch, and target-removed models. Repeat the complete fit for both null
   families.
4. **WP-4: exposed HEST runner — preflight/synthetic smoke complete; biological phase blocked.**
   The source hash, schema, encoder, crop registry, type remapping, and registered blockers were
   verified. The synthetic smoke exercised checkpoint replay and 20 complete refits per null
   family under bounded resources. No HEST molecular row was fitted and no engineering decision
   was issued because the blank-patch and registration-sensitivity requirements cannot run.
5. **WP-5: conditional multiscale token pooling.** Begin only if global embeddings fail geometry or
   mask-specific results point to localization/pooling as the limiting factor. This requires a new
   visible positive-control protocol.
6. **WP-6: development-only nonlinear RNA target.** Begin only if basis ceiling, reconstructed-gene
   variance, or held-out reconstruction shows the linear molecular target is limiting. Report all
   primary results in measured-gene space.

## Track B — scientific authorization infrastructure

In parallel with exposed estimator work:

1. Obtain immutable, target-independent annotation provenance and row-level validation receipts.
2. Complete development-only H-MEAS and freeze a common reliable measured-gene panel.
3. Replace the computationally infeasible fixed calibration with a preregistered sequential design
   using simultaneous confidence bounds.
4. Register a genuinely pristine, cell-resolved cohort from metadata only before molecular outcomes
   are opened.

Estimator work cannot substitute for any of these prerequisites.

## Track C — references and end-to-end inference

This track remains blocked until a pristine H-CELL protocol passes.

1. Test reference fusion v3 with a single donor/type centroid as the primary bank and centered
   donor residual fusion.
2. Require matched banks to beat image-only, type routing, every wrong donor, and a generic bank
   while preserving variance, coverage, and rare-state performance.
3. Add calibrated H&E-only type routing and fixed-abstention evaluation.
4. Consider conditional mean matching, CORAL, or MMD before any adversarial alignment. Unpaired
   alignment remains H-COMP evidence and cannot establish cell-level instance correctness.

## Stop rules

- Nonlinear image models that do not beat the architecture-matched nonlinear nuisance model do not
  support image-specific information.
- Similar focal and target-removed performance blocks a target-cell interpretation.
- Improvement accompanied by molecular variance collapse blocks reference expansion.
- A retrospective pass allows only prospective protocol design; all biological authorization
  fields remain false.
- A failed predecessor is retained in the evidence bundle and is never relabeled.

## Deferred components

Encoder fine-tuning, RNA foundation-model substitution, reference-expanded genes, optimal
transport, adversarial alignment, graph learning, unknown-state modules, and iterative refinement
remain deferred until their immediate predecessor passes under the registered evidence hierarchy.
